このLINEスタンプを第四境界公式LINEアカウント([@055bbvav](https://lin.ee/75EV41s))に送ると、AMGYが調査のため質問をしてきます。施設のサイト等を探索し、答えを見つけてください。

## ヒント・答え

{{% accordion title="スタンプ制作の関係者" %}}
    {{% accordion title="ヒント①" %}}
トップページ > マスコットキャラ > イラストレーター https://kagamino-jrep.net/character/designer/

こちらのページによると、シナイドリをデザインしたイラストレーターはぷりゅまさんです。
    {{% /accordion %}}
    {{% accordion title="ヒント②" %}}
ぷりゅまさんはXアカウント([@pluma_stella](https://x.com/pluma_stella))を持っており、その投稿からLINEスタンプを制作しているのが確認できます。
    {{% /accordion %}}
    {{% accordion title="答え" %}}
ぷりゅま
    {{% /accordion %}}
{{% /accordion %}}

{{% accordion title="彼女のお気に入りのモチーフ" %}}

トップページ > マスコットキャラ > イラストレーター https://kagamino-jrep.net/character/designer/

こちらのページに掲載されています。
    {{% /accordion %}}
    {{% accordion title="答え" %}}
お星様とトリのモチーフ
    {{% /accordion %}}
{{% /accordion %}}

{{% accordion title="仕事場を調べる" %}}
この答えはぷりゅまさんがXに投稿している[Privatterの投稿](https://privatter.net/i/7808588)のパスワードです。英語小文字8文字だということが分かっています。
    {{% accordion title="ヒント①" %}}
ぷりゅまさんのXアカウントのヘッダーが仕事場の絵です。
    {{% /accordion %}}
    {{% accordion title="ヒント②" %}}
ヘッダーには仕事場にあるものの名前が英語で書かれていますが、この内「お星様とトリのモチーフ」がついているのは

* apron
* ink
* envelope
* poster
* penholder
* easel
* lamp
* plant

です。
    {{% /accordion %}}
    {{% accordion title="ヒント③" %}}
頭文字を抜き出し並び替えてみてください。
    {{% /accordion %}}
    {{% accordion title="答え" %}}
applepie
    {{% /accordion %}}
{{% /accordion %}}

{{% accordion title="制作指示書を調べる" %}}
    {{% accordion title="ヒント①" %}}
制作指示書はジグソーパズルになっています。青のピース、ピンクのピースを並び替えてそれぞれパズルを完成させてみましょう。
    {{% /accordion %}}
    {{% accordion title="ヒント②" %}}
各パズルにおいて、並び替えたピースに相当するLINEスタンプを並べてみましょう。
    {{% /accordion %}}
    {{% accordion title="ヒント③" %}}
青ピースにおいて、横書きの要領で対応するLINEスタンプの頭文字を拾って読んでいくと「あおもじよめ」になります。
    {{% /accordion %}}
    {{% accordion title="答え" %}}
ピンクピースにおいて、横書きの要領で対応するLINEスタンプの青文字を拾って読んでいくと「くにまつがどくのます」になります。これをAMGYに伝えます。
    {{% /accordion %}}
{{% /accordion %}}
